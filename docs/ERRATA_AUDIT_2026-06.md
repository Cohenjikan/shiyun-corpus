## 二、全库完备体检(找茬·找漏洞)

- **schema / 朝代键 / 简体编码 / 许可 分层**:`validate.mjs` 全过(✅ PASS)。
- **无古文(散文)泄漏**:prose-title 启发式命中(题尾含 赋/书/碑/传/序…)逐条复核,**全部为咏物/咏史/纪事诗**("赋"为动词"赋诗"、"…碑/传"为题咏对象),**无散文混入**。
- **跨朝代同名作者 1,584 个**:绝大多数为**同名异人**(poetId=`fnv32(name|dynasty)` 已正确区分为不同诗人),少数为上游互见/误收;铁律不归并,交消费者判断。典型:金/宋/元边界(金末元初)同名多。
- **跨作者互见(正文相同、作者不同)293 组 / 611 首**(占 0.06%):`dup_group`(键=作者+正文)**不覆盖**此类。内含**联句**(合著,本就列于各参与者名下,合理)与**争议归属/误收**(如晏殊/苏轼《诉衷情·海棠》、无名氏/秦观《泗州东城晚望》)。已量化报告;消费者若需跨作者去重可按正文哈希自行归并。
- **毛泽东《到韶山》跨源重出 + 跨朝代**:骨干本(jinxiandai,已订正)与 yuxqiu 本(dangdai,保真原文)并存,`dynasty` 因跨源分桶不一致;铁律不归并,记录在案。
- **占位符记录 13,393 首**(正文含 `?`/`�`,多为 utf8mb4 生僻字被上游替换):铁律完整不删字,**保留并报告**,净化交消费者。
